Methods

authorize(clientId, redirectUrl, state)

Initiates a redirect to authorize the client using oauth
Parameters:
Name Type Description
clientId string The application's Client ID
redirectUrl string The redirect URL to return to after authentication. This must be an authorized URL for the client.
state string (Optional) State variable that is returned to the application after authentication. This can be grabbed from the .getState() method.
Source:
  • purecloudsession.js, line 63
Examples
pureCloudSession.authorize('XXXXXXXX-XXXX-XXXX-XXXX-XXXXXXXXXXXX', 'http://localhost:8085/examples/').done(function(){
         //this method will be called once we have a valid authorization token
         // if we don't have one a redirect to login will be called and then after redirecting back here,
         // the done method will be called.
     });
pureCloudSession.authorize('XXXXXXXX-XXXX-XXXX-XXXX-XXXXXXXXXXXX', 'http://localhost:8085/examples/', "State Value");
